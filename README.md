# 贪吃蛇游戏

一个使用原生 HTML/CSS/JavaScript 开发的贪吃蛇游戏。

## 功能

- 经典贪吃蛇玩法
- 计分系统（支持本地存储最高分）
- 暂停功能
- 多难度选择（慢速/中速/快速）
- 响应式设计，支持桌面和移动端
- 支持微信浏览器

## 控制

### 桌面端
- 方向键或 WASD 控制蛇的移动
- 空格键或 ESC 键暂停游戏

### 移动端
- 屏幕下方的虚拟方向键控制移动

## 部署到 GitHub Pages

1. 创建一个 GitHub 仓库
2. 上传 `index.html` 文件
3. 进入仓库的 Settings → Pages
4. Source 选择 `main` 分支
5. 点击 Save
6. 等待 1-2 分钟，游戏就部署完成了
