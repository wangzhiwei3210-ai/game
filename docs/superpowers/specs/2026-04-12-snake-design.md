# 贪吃蛇游戏设计文档

**日期:** 2026-04-12
**项目用途:** 演示作品

---

## 概述

一个使用原生 HTML/CSS/JavaScript + Canvas API 开发的贪吃蛇游戏，单文件实现，部署到 GitHub Pages，支持桌面和移动端（包括微信浏览器）。

## 核心功能

| 功能 | 描述 |
|------|------|
| 基础贪吃蛇 | 经典玩法，吃食物变长，撞墙或撞自己结束 |
| 计分系统 | 显示当前分数，本地存储最高分 |
| 暂停功能 | 游戏状态支持暂停/继续 |
| 多难度选择 | 慢速、中速、快速三个级别 |

## 视觉风格

- **风格:** 现代简洁
- **背景:** 深蓝到紫色渐变
- **配色:**
  - 蛇头: `#4ade80` (亮绿)
  - 蛇身: `#22c55e` (绿)
  - 食物: `#f97316` (橙色)
- **效果:** 圆角、阴影、动画过渡

## 控制方式

- **桌面端:** 键盘方向键
- **移动端:** 虚拟方向按钮（位于画布下方）

## 项目结构

```
game/
└── index.html    # 单文件包含所有代码
```

## 架构设计

### HTML 结构
- `<canvas>` - 游戏画布
- UI 容器 - 分数、难度、暂停按钮
- 虚拟方向键 - 移动端显示

### JavaScript 模块
| 模块 | 职责 |
|------|------|
| `Snake` 类 | 管理蛇的状态和移动 |
| `Game` 类 | 管理游戏循环、状态、碰撞检测 |
| `Renderer` 对象 | 负责绘制画面 |
| `InputHandler` 对象 | 处理键盘和触摸输入 |

### 游戏状态枚举
- `playing` - 游戏进行中
- `paused` - 暂停状态
- `gameover` - 游戏结束

## 响应式设计

| 设备 | 画布尺寸 | 控制方式 |
|------|----------|----------|
| 桌面端 | 600x600 (或适配窗口) | 键盘方向键 |
| 移动端 | 自适应屏幕宽度 (最大 600px) | 虚拟方向键 |

## 游戏流程

```
开始 → 选择难度 → 游戏进行中
                         ↓
                    撞墙/撞自己 → 游戏结束 → 显示分数 → 重新开始
                    暂停 → 暂停菜单 → 继续游戏/重新开始
```

## 部署方式

### GitHub Pages 部署步骤

1. 创建 GitHub 仓库
2. 上传 `index.html` 文件
3. Settings → Pages
4. Source: `main` 分支
5. Save，等待 1-2 分钟
6. 获取访问链接

## 技术要点

- `requestAnimationFrame` - 流畅动画
- `localStorage` - 最高分持久化
- `resize` 事件 - 响应式适配
- CSS 媒体查询 - 桌面/移动端区分

## 难度参数

| 难度 | 速度 (ms/帧) |
|------|-------------|
| 慢速 | 150 |
| 中速 | 100 |
| 快速 | 60 |

## 数据持久化

- `localStorage.getItem('snake_highscore')` - 获取最高分
- `localStorage.setItem('snake_highscore', score)` - 保存最高分
